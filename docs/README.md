# Document Corpus

This directory contains a curated set of synthetic but realistic
enterprise documents used to demonstrate a Retrieval-Augmented
Generation (RAG) assistant.

## Document Types
- Security control summaries (NIST-aligned)
- Internal policies
- Operational runbooks

## Design Intent
The documents are intentionally:
- Small and readable
- Representative of regulated environments
- Safe for public sharing

They serve as a stand-in for proprietary internal documentation.
